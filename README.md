# Solid Plausible

Plausible provider for your [SolidJS](https://solidjs.com) app.

- [Usage](#usage)
- [License](#license)

## Usage

Install with [pnpm](https://pnpm.io/) (`solid-js` should be installed as a dev dependency):

```sh
pnpx jsr add -D @strootje/solid-plausible
```
